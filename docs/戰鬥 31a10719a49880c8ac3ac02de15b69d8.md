# 戰鬥系統：無上限靈力負載與瞬間推演

## 🌟 系統核心定位

《鏡界》屏除了傳統 RPG 複雜的動作操作與固定的「裝備欄位（頭盔/衣服/武器）」。

本系統採用**「戰前資源構築（Loadout） ➔ 瞬間數值推演 ➔ 文字日誌結算」**的模式。玩家比拼的是「戰前準備（境界、五行、裝備積累）」與「資源分配」，完美契合單手遊玩、瞬間回饋的 LBS 手遊節奏。

---

## 🔋 一、 核心機制：無上限靈力負載 (Cost 系統)

打破傳統裝備格數限制，將修仙者的個人素質極簡化為一個**「靈力電池」**。裝備本身提供攻防數值，而玩家的靈力決定能「驅動」多少裝備。

- **【靈力上限（最大負載）】：**
    - 來源：隨「境界」突破大幅提升（如：煉氣初期 100 點 ➔ 築基初期 1000 點）。
    - 這是玩家出戰配置的**「唯一物理限制」**。
- **【無上限配置槽】：**
    - 玩家可自由攜帶「任意數量、任意種類」的法寶與符籙上陣。
    - **唯一規則：** 所有出戰物品的「靈力消耗（Cost）總和」**絕對不可超過**玩家當前的靈力上限。

---

## 🎒 二、 戰備物資分類（三大流派核心）

玩家可根據自身的財力與庫存，在靈力上限內自由混搭以下物品：

1. **法寶 / 兵器（常駐裝備）**
    - **特性：** 永久存在，提供高額且穩定的單發攻擊力或防禦力。
    - **負載：** **極高。**（例如：【青淵劍】佔用 80 點靈力，攻擊力 800）。
    - **定位：** 境界達標後的主力輸出，用來突破高等妖獸的防禦。
2. **符籙 / 暗器（一次性消耗品）**
    - **特性：** 戰鬥結算後**直接銷毀**（不論勝負）。
    - **負載：** **極低。**（例如：【爆炎符】佔用 10 點靈力，攻擊力 500）。
    - **定位（強大付費點）：** 課長專屬的「鈔能力」流派。遇到打不過的 Boss，可消耗大量靈石購買符籙，將剩餘的靈力全部塞滿爆炸符籙，瞬間轟殺大妖！
3. **功法 / 陣法（被動光環）**
    - **特性：** 提供全局 Buff，如「水系傷害 +30%」或「每回合抵消 200 點傷害」。佔用中等靈力。

## 💥 三、 瞬間推演的三大底層邏輯

當玩家點擊「發起攻擊」並扣除【體力】後，伺服器將在 0.1 秒內跑完以下三層數值碰撞：

### 1. 護體真氣（破防機制 / 防止低等道具堆屍）

- **機制：** 每隻妖獸都擁有固定的「護體真氣（減傷值）」。
- **公式：** `單次有效傷害 = 裝備攻擊力 - 妖獸護體真氣`。（若結果小於 0，強制只扣 1 滴血）。
- **意義：** 防止玩家帶 100 張「傷害 50」的新手符籙，去秒殺「護體真氣 200」的高等妖獸。逼迫玩家追求高單發傷害的極品法寶。

### 2. 五行相剋（戰略核心）

- **公式：** 金剋木、木剋土、土剋水、水剋火、火剋金。
- **機制：** 若玩家出戰的法寶屬性剋制妖獸，該法寶最終傷害 **+30%**；若被剋制，自身受到傷害 **+30%**。結合 LBS 地利，這是無課玩家逆襲的關鍵。

### 3. 境界壓制（絕對力量）

- **機制：** 修仙界等級森嚴。築基期打煉氣期妖獸，基礎傷害直接附加 **+50% 境界壓制加成**。逼迫玩家乖乖花時間去「打坐掛機」提升境界。

## 📜 四、 戰鬥結算與殘酷代價

戰鬥瞬間運算完畢，UI 畫面會彈出類似「修仙模擬器」的文字日誌跑馬燈。

- **🟢 勝利結算（斬殺妖獸）：**
    - **日誌範例：** *「你祭出【青木劍】，藉著屬性相剋死死壓制了【赤炎虎】！經過一番激戰，成功將其斬殺！」*
    - **結果：** 扣除體力、消耗品（符籙）銷毀。獲得妖丹、法寶、五行素材。
- **🔴 失敗結算（重傷遁走）：**
    - **日誌範例：** *「【赤炎虎】發出一聲怒吼，震碎了你的護體真氣！你身受重傷，燃燒精血勉強逃生...」*
    - **殘酷代價：** 扣除體力、消耗品銷毀。因燃燒精血，**直接扣除 1~3 年壽元（生命上限）！** 若壽元歸零則觸發死亡輪迴。

## 💡 五、 UI 介面呈現建議 (開發者筆記)

- **戰前配置介面（購物車模式）：**
    
    不需要畫傳統裝備格子。畫面上方顯示一條大大的**「靈力負載條 (例如：850 / 1000)」**。下方為玩家的物品列表，玩家可無限點擊添加物品上陣，只要頂部的負載進度條不爆表即可發起攻擊。
    
- **戰鬥過程介面（文字 MUD 模式）：**
    
    不需要 3D 動畫或繁雜特效。只需播放帥氣的法術音效，配合由下往上捲動的「戰鬥文字日誌」，極具修真小說的沉浸感，同時極大節省單人開發的美術成本。

## 六、 凡人武鬥系統（受限的戰鬥體驗）

為了凸顯未來成為修仙者後「無上限靈力槽」的自由與爽快感，凡人期的戰鬥必須是**「充滿限制的」**。

### 1. 戰備配置（固定三格位）

沒有靈力，凡人只能靠雙手和世俗兵器。系統介面在凡人期會**隱藏靈力條**，並鎖死只能裝備三樣物品：

- **【主武器】：** 鐵劍、大刀、長棍（提供基礎物理攻擊力）。
- **【防具】：** 皮甲、鐵鎧（提供物理減傷）。
- **【暗器/消耗品】：** 飛鏢、石灰粉、金創藥（凡人版的一次性道具，傷害極低）。

### 2. 戰鬥推演邏輯（關閉五行與境界）

凡人打架就是單純的肉搏，沒有魔法。

- **關閉五行相剋：** 凡人武器沒有屬性，打野狗也不看五行。
- **屬性來源：** 玩家的有效戰力 = `(主武器攻擊力) + (目前體魄 % * 基礎臂力)`。體魄掛機掛得越高，打人越痛。

### 3. 對手設計（街頭巷戰）

- 地圖上掃描出的不再是「大妖」，而是「地痞流氓」、「發狂的野狗」或「攔路山賊」。
- 戰鬥日誌呈現濃厚的武俠風：
    
    > *「你抽出【生鏽的鐵劍】，一招『長拳』直逼【攔路山賊】面門！」「山賊揮舞大刀反擊，砍在你的【粗布衣】上，你受了點皮肉傷。」「你找準破綻，一劍刺穿山賊，戰鬥勝利！獲得銀兩 50、山賊的頭巾。」*
    >

## 七、 鯉魚躍龍門：覺醒之戰 (Boss 戰)

當玩家的【體魄】與【真氣】皆掛機達到 100% 時，就會觸發凡人期的終極考驗，這也是引導他們進入修仙系統的震撼教育。

- **強制事件：** 玩家在 LBS 採集時，必定會遭遇一隻**「一階妖獸（如：變異火狐）」**。
- **數值輾壓：** 凡人武器打在妖獸的「護體真氣」上幾乎無效（全部只扣 1 滴血）。
- **劇情殺與覺醒：** 戰鬥日誌會顯示玩家陷入絕境。此時，體內 100% 的真氣與體魄產生共鳴，系統會**強制贈送並自動裝備一張【新手體驗版・天雷符】**。
- **跨越階級的一擊：** 系統引導玩家使用這張符籙，無視三格位限制，瞬間爆發出凡人不可能擁有的龐大靈力，將火狐秒殺！
- **結算：** 獲得「妖獸內丹」，系統彈出大字：
    
    > *「凡根褪去，靈脈初醒！你正式踏入【煉氣期初期】！」(UI 介面瞬間切換：固定三格位碎裂，『無上限靈力槽』正式解鎖！)*
    >

### 👨‍💻 開發者實作筆記 (API 共用策略)

這套系統最大的好處是**「不需要寫兩套戰鬥邏輯」**。
在後端 API 裡，凡人和修仙者跑的是同一支結算程式。你只需要在凡人的裝備（鐵劍、皮甲）設定上：

- `Lingli_Cost = 0`（不需要靈力驅動）
- `Element = None`（無五行屬性）

而在前端 UI 裡，寫一個簡單的判斷式：`if (Player.Stage == 'Mortal')`，就隱藏靈力條，並強制只顯示三個裝備框。一旦升上煉氣期，就解除 UI 限制。這樣既省開發時間，又做足了遊戲體驗的對比與儀式感！
