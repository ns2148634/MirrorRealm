# 情報

**這套系統完美實現了單人開發的技術降維，同時保留了修仙界「探索、交易、奪寶」的核心樂趣。無課玩家靠「腳」賺錢，課長靠「靈石」買時間與空間。**

# **🗺️ 一、 產出端：凝結秘境陣盤（無課玩家的淘金）**

- **行為： 玩家親自移動到特殊 LBS 地標（如台北 101、北投溫泉、深山廟宇），消耗大量【精力（神識）】進行深度掃描。**
- **機制（無須花費靈石）： 掃描成功後，系統將該地標的特殊機緣具象化，生成一枚實體的**【地標專屬・秘境陣盤】**（存入背包）。**
- **例如：【雷劫・通天塔陣盤】（綁定座標：台北 101）。**
- **設計意義： 鼓勵玩家出門跑圖。因為打包不扣靈石（只扣隨時間恢復的精力），這確保了市場上永遠有源源不絕的商品供應。**

# **🛍️ 二、 交易端：萬象坊市與 10% 天道稅**

- **上架與定價： 探索玩家將【秘境陣盤】放上拍賣場，由玩家自由定價（例如標價 1,000 靈石）。**
- **無風險交易： 因為陣盤已經是實體道具，買家在市場上看到的就是確定收益，不會有「買了跑過去發現怪被打死」的詐騙感。**
- **天道稅（經濟回收）： 交易成功時，系統強制抽取 10% 靈石 作為手續費。賣家實拿 900，100 靈石被系統永久銷毀（完美抗通膨）。**

# **🚀 三、 消費端：跨界破陣（官方最大的靈石收入來源）**

**這是系統最毒辣、最賺錢的環節！買家（課長）買到了陣盤，但他人在桃園，陣盤綁定的卻是台北 101 的座標。**

**此時，課長面臨兩個選擇：**

**1. 肉身前往（耗費大量體力與現實時間）： 真的搭車去台北 101，站在那裡點擊陣盤開啟副本。（通常大課長沒這個美國時間）。**

**2. 神識降臨（官方付費點）： 課長選擇坐在家裡的沙發上，直接開啟副本。**

- **【強烈逼課點】： 系統提示：「此秘境陣盤綁定於『台北 101』，距離過遠，需消耗一張【高級空間傳送符】方可神識降臨破陣！」**
- **課長立刻點開官方商城，花費 100 靈石 買下一張【空間傳送符】，瞬間開啟專屬的單機副本，安心打王拿極品裝備。**

# **💰 四、 經濟生態閉環總結（雙重提水）**

**在這一次的交易循環中，達成了完美的生態平衡：**

**1. 探索者（賣家）： 出了勞力跑 101，沒花半毛真錢，賺到了 900 靈石，可以拿去買壽元丹續命，對遊戲充滿黏著度。**

**2. 大能（買家）： 沒出門，花了 1,000 靈石買陣盤，又花了 100 靈石買傳送符。雖然花了 1,100 靈石，但 100% 穩拿極品裝備，獲得了「用錢買特權」的極致爽感。**

**3. 官方（你）： 在這一次交易中，你回收了 100 靈石（10% 交易稅） + 100 靈石（傳送符的官方商城直購） = 200 靈石的淨回收！**

**💡 開發者設計建議**

**把收費點放在**「開啟異地副本的瞬間（傳送符）」**是最高明的做法。**

- **UI 呈現： 當玩家在背包點擊買來的異地陣盤時，畫面跳出一個很帥的「虛空裂縫」動畫，提示距離太遠，並直接彈出「是否花費 100 靈石使用空間符？」的快捷購買按鈕。這能創造極高的衝動消費轉化率。**
- **技術實現： 單純的數值扣減與開啟單機戰鬥介面，沒有任何連線同步問題，一個人完全可以輕鬆寫完！**
